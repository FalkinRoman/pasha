# Референсы конкурентов

Сюда складываем скрины приложений компьютерных клубов, по которым делаем stopkek.

## Папки

| Папка | Что класть |
|-------|------------|
| `mobile/` | Мобилка: карта зала, бронь, профиль, оплата, таймер, двери |
| `pc-launcher/` | Оболочка на ПК: вход, игры, лаунчеры, тех-чат |
| `admin/` | Админка (web / desktop) |
| `other/` | Всё остальное: сайт, TG-бот, вывеска, тарифы |

## Именование файлов (удобно для разбора)

```
<клуб>_<экран>_<комментарий>.png
```

Примеры:
- `tss-game_floor-plan_main.png`
- `colizeum_booking_payment.png`
- `unknown_launcher_games-menu.png`
