# Скриншоты App Store

## Размер (iPhone 6.5")

**1284 × 2778 px** — портрет, раздел «Скриншоты iPhone 6.5 дюйма» в App Store Connect.

## Какие экраны заливать (порядок важен — первые 3 видны в поиске)

| # | Файл | Зачем |
|---|------|--------|
| 1 | `01-welcome.png` | Бренд, ценность приложения |
| 2 | `02-map.png` | Главная фича — выбор капсулы |
| 3 | `03-home.png` | Баланс + активная бронь |
| 4 | `04-session.png` | Управление сеансом, дверь, таймер |
| 5 | `05-login.png` | Вход по звонку (опционально) |

Минимум для ревью: **3 скриншота**. Рекомендуем **5**.

## Сгенерировать PNG

```bash
cd stopkek-mobile/store-assets
npm install
npx playwright install chromium
npm run generate
```

Результат: `output/iphone-65/*.png`

## Загрузка в App Store Connect

1. [App Store Connect](https://appstoreconnect.apple.com/apps/6778068340/distribution/ios/version/inflight)
2. **Скриншоты** → iPhone 6.5"
3. Перетащи PNG из `output/iphone-65/` в нужном порядке

## Рекламный текст (до 170 символов)

```
Забронируй соло-капсулу в компьютерном клубе. Вход по звонку, таймер сеанса и управление дверью — в одном приложении.
```

## Описание (кратко)

```
стопКЕК — мобильное приложение компьютерного клуба с закрытыми соло-капсулами.

• Выбор места на карте зала
• Бронь и оплата с баланса
• Вход в клуб по звонку
• Таймер сеанса и открытие двери
• Продление игры из приложения
```

Макеты в `screens/*.html` — правь верстку и перегенерируй.
