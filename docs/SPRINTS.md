## Детализация спринтов

### S-00 · Аналитика и проектирование
**1–1.5 нед · 20 000 ₽ · ПРЕДОПЛАТА (этап «Старт»)**

| Задача | Результат |
|--------|-----------|
| Финализация ТЗ | [TECHNICAL_SPEC.md](./TECHNICAL_SPEC.md) |
| UX/UI в Figma | Ключевые экраны mobile, admin, launcher (wire → UI) |
| Архитектура БД | ER-диаграмма, миграции v0 |
| API контракты | OpenAPI / Swagger черновик |
| CI/CD + VPS | Docker-compose, staging, домен, SSL |
| Схема зала stopkek | SVG-координаты мест (по плану клуба) |

**Критерий сдачи:** утверждённые макеты + репозитории + поднятый staging API `health`.

---

### S-01 · Backend Core
**2 нед · 70 000 ₽ · ЭТАП 1**

| Задача | Результат |
|--------|-----------|
| NestJS scaffold | модули, guards, config |
| PostgreSQL | users, zones, seats, bookings |
| Auth SMS OTP | send/verify, JWT, Redis sessions |
| CRUD компьютеров | зоны, места, hardware specs |
| WebSocket | `seat.status` broadcast |
| Бронирование v1 | create, conflict check, statuses |
| Docker deploy | staging/prod |

**Критерий сдачи:** Postman/ Swagger: регистрация → создание брони; WS меняет статус места.

---

### S-02 · Mobile App MVP
**2–3 нед · 100 000 ₽ · ЭТАП 1**

| Задача | Результат |
|--------|-----------|
| Expo + RN setup | iOS/Android, Redux Toolkit |
| Auth flow | телефон → OTP → профиль |
| Карта зала | SVG/Canvas, zoom, real-time WS |
| Бронирование UI | выбор места, время, расчёт (без оплаты или mock) |
| Профиль | баланс (read), история броней |
| Дизайн-система | тёмная тема, красный акцент, шрифты по референсу |

**Критерий сдачи:** TestFlight / internal APK: полный UX до экрана оплаты.

**Зависимости:** S-01 API готов.

---

### S-03 · Оплата и финансы
**1 нед · 15 000 ₽ · ЭТАП 1 (закрывает этап 1)**

| Задача | Результат |
|--------|-----------|
| YooKassa | topup, webhook, idempotency |
| Кошелёк | balance, transactions |
| Оплата брони | списание / оплата картой |
| Таймер сеанса | server time, `active` booking |
| Продление | extend API + mobile UI |
| Mobile | экраны пополнения и оплаты |

**Критерий сдачи:** E2E: бронь → оплата → таймер → продление на staging.

**Этап 1 закрыт:** 171 500 ₽ (S-01 + S-02 + S-03).

### S-04 · Умные двери и приёмка
**1 нед · 15 000 ₽ · ЭТАП 2**

### S-05 · PC Launcher (заглушка)
**2 нед · 100 000 ₽ · ЭТАП 2**

| Задача | Результат |
|--------|-----------|
| Electron kiosk | fullscreen, autostart |
| Auth QR/PIN | связь с booking/session |
| UI «Игры / Платформы» | Steam, Epic, Discord (deeplink/shell) |
| Тех-чат | WS → admin/TG |
| Session lock | блок при end |
| Дизайн | стиль stopkek / лучший референс с обхода клубов |

**Критерий сдачи:** установка на 1 тестовый ПК; вход → меню → блокировка по таймеру.

> **Примечание:** финальный UI launcher — после сбора референсов с обхода клубов; в рамках спринта — рабочий MVP.

## Этап 1 / 2 / 3 — что отдаём заказчику

### Этап 1 — «Клиент может бронировать и платить» (S-00…S-03)
- Staging + prod-ready backend
- Mobile: регистрация, карта real-time, бронь, оплата, таймер, продление
- **Не входит:** двери, launcher, admin, push, сторы

### Этап 2 — «Клуб работает под ключ» (S-04…S-06)
- Двери + приёмка
- PC Launcher на ПК
- Admin + Telegram
- **Не входит:** push, публикация в сторах

### Этап 3 — «Релиз» (S-07…S-08)
- Push-уведомления
- Финальный polish
- QA + App Store / Google Play + документация
